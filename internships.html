Two Internships Available

Two internships are available at Yale-NUS College, Singapore, supervised by Dr. Anthony Widjaja Lin. A successful candidate will have an opportunity to work on one of the following topics:

  • Program analysis for improving web performance (e.g. detecting redundant CSS rules in HTML applications).
  • Program analysis for detecting security vulnerabilities in web applications (e.g. using string solving)
  • Formal modelling and analysis of computational models in biology and ecology.

A successful candidate will have an opportunity to work on either foundational issues (including decidability, algorithmic complexity), or implementation issues (including benchmarking, and optimisation), or both.

An applicant should either be pursuing, or have completed, an undergraduate degree in computer science (or a related area). Applications will be reviewed on a first-come, first-serve basis. Please email Dr. Anthony Widjaja Lin for more information. Enquiries or expression of interests are very welcome.
